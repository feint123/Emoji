##说明
1. code
    200：正常
    404: 未能与服务器君产生联系
    111：用户不存在
    112：不存在该分类
    113：该标签不存在
    121：发布的消息中存在非法词汇
    122：发布的消息不符合规范

2. TEXT
        [    
             {
                tid:bigint,
                tname:string(1,50),
                tip:string(35,150),
                tag:[{0:string(1,10)},{1:string(1,10)}],
                date:yyyy-MM-dd,
                collected:boolean,   ##收藏
                support:int,    ##点赞
                hot:int     ##热度：阅读量占总用户数的比例
             } 
        ]
